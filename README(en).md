# Propofol Education Demo

This is a simple educational simulator designed for trainees to understand 
the relationship between propofol effect-site concentration (Ce) and anesthetic depth (BIS).

## Purpose
This tool helps visualize:

- How anesthetic sensitivity changes with age (EC50 shift)
- The difference between induction and emergence (hysteresis)

It is intended for building intuitive understanding of anesthesia depth.

## Important Notice
This simulator is for educational purposes only.

- It is NOT a clinical tool
- It must NOT be used for medical decision-making
- The model is simplified and does not represent individual patients

## How to Use
Open the HTML file in a browser or access via GitHub Pages.

Adjust:
- Age
- Sex
- BMI

and observe how the Ce–BIS curve changes.

## Target Users
- Medical trainees
- Residents learning anesthesia basics
