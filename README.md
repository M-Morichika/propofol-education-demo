# プロポフォール麻酔深度シミュレータ（教育用）

本ツールは、プロポフォールの効果部位濃度（Ce）と
麻酔深度（BIS）の関係を直感的に理解するための
教育用シミュレータです。

## 目的

研修医・初学者が以下の概念を視覚的に理解することを目的としています：

- 年齢による麻酔感受性の変化（EC50のシフト）
- 導入と覚醒の違い（ヒステリシス）

## 使用方法

ブラウザでHTMLファイルを開くか、
GitHub Pages上で動作します。

以下を変更できます：

- 年齢
- 性別
- BMI

これにより、Ce–BIS関係の変化を確認できます。

## 注意

本ツールは教育目的の簡略モデルです。

- 臨床判断には使用できません
- 実際の患者データを反映したものではありません
- 個別症例の予測には使用できません

## 対象

- 研修医
- 麻酔を学ぶ初学者
